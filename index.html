Day 1

Intrduction to task, December 11, 2017

We have been given a Turing machine simulator and have been set the task of improving the functionality of the program.

Our initial thoughts are to troubleshoot any bugs that are there within the program.

At first we ran into issues when trying to load the website on our local machines, which slowed our progress.

Once we overcame this problem, we fixed minor bugs such as the 'Decimal-to-Binary' text file, the drop-down save menu and the popup error messages.

Day 2

Tidy up button and login, December 12, 2017

As an alternative to the current hash ID creation when a user wants to save a custom program, we decided to create a login based system. the user will enter a username and password- if it is correct they will be logged in and if they are not yet a member they will automatically have an account created with the username and password entered.
