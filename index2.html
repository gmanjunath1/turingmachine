Day 3

Editing tape pointer, December 13, 2017

We were told that a good feature to implement would be for the user to choose how they want the tape to move when it is being output.

We added code which put a drop down menu in the website allowing the user to choose where they wanted the tape to be positioned.

We have given them the following options: Dont move, Move with head, Move if necessary, Stick to left, and Stick to right.
